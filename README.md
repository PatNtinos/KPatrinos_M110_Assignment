# Dungeon Keeper: Unity Engine

A small management-style game built in **Unity** with custom **C# gameplay scripts**.  
Manage an underground dungeon where you clean, feed creatures, maintain rooms, and keep the lair running smoothly.

## Project Info

This game was created as a **small project** for the *Interactive Systems* course in Unity.  
The goal was to **experiment with Unity and C# integration** while building a simple management-style game.  
It is meant for learning and exploration.
